CRM.WG plugin enables you to add Javascripts of CRM.WG system or any other scripts to WordPress footer.

== Description ==
Плагин был разработан для клиентов системы CRM.WG  с целью максимально упростить добавление кода системы
на сайтах созданных на системе управления контентом Wordpress.

Данный плагин вставляет код CRM.WG в футер сайта.
Кроме вставки кода ссистемы CRM.WG ,плагин можно использовать и по другому назначению,например для вставки своего кода
на языках HTML,CSS,JS  в футер сайта --`<footer>`.


= Преимущества плагина: =

* Плагин дает возможность правильно и легко подключить код CRM.WG в сайте ,который использует систему управления Вордпресс.

* Также плагин можно использовать для вставки исключительно своего кода на языках HTML,CSS,JS(например,счетчики посещений итд)  в футер сайта --`<footer>`.

При возникновении каких либо трудностей в использовании плагина,пожалуйста,обращайтесь по ссылке ниже
 https://widgetgen.com/kontakty/

== Installation ==

Загрузить плагин  по директории `/wp-content/plugins` и активировать.Или найти плагинв консоли по адресу "CRM.WG" from `Консоль --> Плагины --> Добавить новый`


== Screenshots ==

1. Плагин можно найти в поиске по автору widjetgen
2. Для использования и настройки плагина, нажмите кнопку "Настроить"
3. Вставьте  CRM.WG код в это окно
